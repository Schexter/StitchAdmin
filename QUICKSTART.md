# StitchAdmin 2.0 - Schnellstart

## Erste Schritte nach der Installation

### 1. Anmeldung
- Öffnen Sie StitchAdmin im Browser: `http://localhost:5000`
- Melden Sie sich mit den bei der Installation erstellten Zugangsdaten an

### 2. Firmeneinstellungen prüfen
- Gehen Sie zu **Einstellungen** → **Firmeneinstellungen**
- Überprüfen Sie Ihre Firmendaten (Name, Adresse, Steuernummer)
- Diese Daten erscheinen auf allen Dokumenten

### 3. Erste Schritte
1. **Kunden anlegen**: Stammdaten → Kunden → Neuer Kunde
2. **Artikel importieren**: Stammdaten → Artikel → L-Shop Import
3. **Erster Auftrag**: Aufträge → Neuer Auftrag

---

## Backup erstellen

**Wichtig:** Erstellen Sie regelmäßig Backups!

1. Klicken Sie auf Ihr Benutzersymbol (oben rechts)
2. Wählen Sie **Updates & Backups**
3. Klicken Sie auf **Jetzt Backup erstellen**

---

## Update installieren

1. Laden Sie das Update-Paket herunter (`.zip`-Datei)
2. Gehen Sie zu **Updates & Backups** → **Update hochladen**
3. Wählen Sie die ZIP-Datei aus
4. Bestätigen Sie die Installation
5. Starten Sie StitchAdmin neu

Bei Problemen: Nutzen Sie die **Rollback**-Funktion in der Backup-Verwaltung.
